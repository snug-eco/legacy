Ed(itor)

Ed (snug) is a command line based editor,
similar to Ed (unix). Ed (snug) operates
directly on the file on disk.
Thus there is now write command.

Commands

q       - quit
g{line} - goto line by moving cursor
i       - insert after line and advance cursor
d       - delete line
c       - change line
n       - enumerate entire file
p       - print line under cursor
